## Format

| 类型  | 大小  | 名称  | 注释  | 其他  |
| --- | --- | --- | --- | --- |

## SPS

### header

| byte   | 0x1 | (头)标识符     | 始终0x48      |           |
| ------ | --- | ---------- |:-----------:| --------- |
| byte   | 0x3 | 块大小        | header的总大小  |           |
| byte   | 0x1 | (块)标识符     | 始终0x14      |           |
| byte   | 0x1 | 值(转换后为通道数) | (值/4)+1=通道数 | 值=0即通道数=1 |
| ushort | 0x2 | 采样率        |             |           |
| byte   | 0x1 | (块)标识符     | 始终0x40      |           |
| Uint32 | 0x3 | 采样点数       |             |           |

### chunk

| byte   | 0x1 | (块)标识符       | 始终0x44 |                      |
| ------ | --- | ------------ | ------ | -------------------- |
| Uint32 | 0x3 | (当前)块的大小     | 数据大小+8 |                      |
| Uint32 | 0x4 | (当前块)解码后数据大小 |        | 需要在编码前写入，编码后无法获取正确大小 |
|        |     | (当前)块数据      |        |                      |

### end chunk

| byte | 0x1 | (块)标识符 | 始终0x45 |     |
| ---- | --- | ------ | ------ | --- |
| byte | 0x3 | 当前块总大小 |        |     |
|      |     | data   |        |     |

## SNR

### header

| byte   | 0x1 | (头)标识符 | 始终0x4       |                        |
| ------ | --- | ------ | ----------- | ---------------------- |
| byte   | 0x1 | 值      | (值/4)+1=通道数 |                        |
| ushort | 0x2 | 采样率    |             |                        |
| Uint32 | 0x4 | 采样点数   |             | 也可能是0x1的标识符,0x3大小的采样点数 |

### chunk

| Uint32 | 0x4 | 块大小     | 数据大小+8  |                      |
| ------ | --- | ------- | ------- | -------------------- |
| Uint32 | 0x4 | 解码后数据大小 | 解码后数据大小 | 需要在编码前写入，编码后无法获取正确大小 |
|        |     | 数据      |         |                      |
